# GIT 操作指南

### 拷贝项目: `git clone <仓库地址>`

### 创建分支: `git branch <name>`

### 进入分支(没有则自动创建):`git checkout -b <分支名>`

### 查看状态: `git status`

### 添加修改: `git add <文件名>`(使用 `.` 作为文件名可添加所有)

### 提交: `git commit -m"提交的描述文字"`

### 拉取: `git pull`

### 推送: `git push`

查看分支列表: `git branch -list`

查看所有分支(含远程): `git branch -a`
